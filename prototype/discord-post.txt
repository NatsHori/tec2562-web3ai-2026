【バグ】デスクトップがアイコンで埋め尽くされている（顧客：自分自身）
【プロダクト】ワンクリックで賢くファイルを自動分類する「Smart Desktop Cleaner」
【動くプロトタイプ】自分のPCのフォルダを選択すると、実際のファイル名がデスクトップアイコンとして表示され、ボタン一つで適切なカテゴリ（Docs, Images, Projects等）へスッと吸い込まれる体験ができるデモWebアプリを作りました！
さらに、裏側では**最先端の爆速AI「Groq API (Llama 3)」がファイル名の意味を汲み取って分類し、賢くサブフォルダ名を名付けてくれる**次世代の整理体験を実装しています！

# プロトタイプの動かし方
1. `npm install`
2. `npm run dev`
3. 画面の「📁 対象フォルダを選択」からPC内の適当なフォルダを選択
4. 実際のファイル名が表示されたら、「✨ Auto Clean Up」ボタンを押してAIのアニメーションを体験
5. 整理されたフォルダをクリックして、AIが名付けたサブフォルダ分類を確認してみてください！

※APIキー入力欄を空欄のまま実行すると、自動的に「設定不要のAIシミュレーション（デモモード）」が起動し、誰でもすぐに賢い分類を体験できます。
※ご自身の Groq API Key を入力すれば、Viteプロキシ経由で本物の Llama 3 (70B) による爆速リアルタイム分類がテスト可能です！
