【バグ】デスクトップがアイコンで埋め尽くされている（顧客：自分自身）
【プロダクト】ワンクリックで賢くファイルを自動分類する「Smart Desktop Cleaner」
【動くプロトタイプ】自分のPCのフォルダを選択すると、実際のファイル名がデスクトップアイコンとして表示され、ボタン一つで適切なカテゴリ（Docs, Images, Projects等）へスッと吸い込まれる体験ができるデモWebアプリを作りました！
さらに、今回は**本物のAI（Gemini 1.5 Flash）を裏側で連携**させています！単純な拡張子や文字の一致ではなく、AIが「ファイル名の意味」を汲み取って分類し、賢くサブフォルダ名を名付けてくれる次世代の整理体験を実装しました！

# プロトタイプの動かし方
1. `npm install`
2. `npm run dev`
3. 画面上部に取得した Gemini API Key を入力
4. 画面の「📁 対象フォルダを選択」からPC内の適当なフォルダを選択
5. 実際のファイル名が表示されたら、「✨ Auto Clean Up」ボタンを押してAIの分析とアニメーションを体験
6. 整理されたフォルダをクリックして、AIが名付けたサブフォルダ分類を確認してみてください！
